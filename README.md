# Hello this is, Magic 8 Cube!
By: Ryan Ni

# Introduction
Magic 8 Cube is a "Magic 8 Ball" inspired project, except better because it's a cube, the superior shape.

# What it does
Magic 8 Cube holds the power of premenision and foresight! But all power comes at a cost, and for Magic 8 Cube that is in the form of yes or no questions.

# How I built it
I programmed a Arduino NANO and powered by a 9 Volt Alkaline Battery.

# Challenges we ran into
One of the chelleges we faced was designing the circuitry for Power Tag to allow for both Solar charging and general charging via a USB Type C Connector. With some more research we decided to change what charging board we would use in order to allow for both Type C and Micro USB inputs and 2 5V 2.4A USB Ouputs.
